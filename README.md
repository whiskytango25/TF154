# TF154

TF154 / Reaper Six project repository.

This repo contains the current working files from the TF154 project folder, including:

- Task Force 154 website files
- Reaper Six book, cover, and reader assets
- Brand, favicon, image, audio, and document assets
- Knowledge base and editorial materials
- Kickstarter and marketing copy

## Project Notes

- Local Claude settings, `.DS_Store` files, secrets, account exports, generated output, and large zip archives are intentionally excluded from Git.
- Vectored Compliance is excluded because it belongs in its own WTI project lane.
- Large binary creative assets are included where they are part of the active TF154 project.
